A RESTful framework that abstracts away a lot of the hassle of creating and maintaining a real API.

Phobos.js can get your API running quickly and efficiently - with its robust standardized format, it can be a great way to get your next app going fast. Simply, Phobos.js removes the annoying complexity of common API tasks so you can focus on what matters most - the client application. Read more in the getting started guide.
